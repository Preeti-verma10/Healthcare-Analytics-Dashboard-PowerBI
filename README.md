# 🏥 Healthcare Analytics Dashboard using Power BI

## 📌 Project Overview

The **Healthcare Analytics Dashboard** is an interactive Power BI project designed to analyze and visualize hospital data for better operational and financial decision-making. The dashboard provides comprehensive insights into patient records, doctor performance, appointments, medicine inventory, and financial metrics through an intuitive and user-friendly interface.

---

## 🎯 Project Objectives

- Monitor patient information and healthcare records.
- Analyze doctor performance and appointment details.
- Track medicine inventory and supplier information.
- Visualize hospital financial performance.
- Provide interactive dashboards for data-driven decision-making.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel

---

## 📊 Dashboard Pages

### 🏠 Home Dashboard

The Home page serves as the central navigation hub, providing quick access to all dashboard sections through an interactive and user-friendly interface.

**Features**
- Navigation buttons
- Dashboard overview
- Interactive layout

---

### 👤 Patient Dashboard

The Patient Dashboard provides a comprehensive view of patient records, diagnoses, admission details, medicine usage, and assigned doctors. It helps monitor patient care and treatment information efficiently.

**Key Features**
- Patient Details
- Assigned Doctor
- Diagnosis Information
- Admission Date
- Medicine Quantity
- Medicine Sales Analysis
- Interactive Charts
- Patient Status

---

### 👨‍⚕️ Doctor Dashboard

The Doctor Dashboard analyzes doctor information, qualifications, appointments, salary details, and patient feedback. It helps evaluate doctor performance and healthcare service delivery.

**Key Features**
- Doctor Information
- Qualification
- Experience
- Salary Details
- Appointment Records
- Patient Feedback
- Payment Status

---

### 💰 Finance Dashboard

The Finance Dashboard provides insights into hospital finances, medicine inventory, supplier performance, and operational costs. It enables efficient monitoring of financial and inventory-related metrics.

**Key Features**
- Medicine Inventory
- Supplier Analysis
- Stock Availability
- Hospital Billing
- Cost Analysis
- Financial Overview
- Staff Statistics
